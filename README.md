# 🐉 Mini RPG Game

A beginner-friendly **Python text-based RPG battle game** where you play as a hero and fight against a powerful dragon. Use attacks, healing, and special moves to defeat your enemy.

## 🎮 Features

- ⚔️ Normal attack system
- 💚 Healing system
- 🔥 Special attack ability
- 🐉 Dragon boss battle
- ❤️ Player and enemy health system
- 🎲 Random damage generation
- ⭐ XP reward system
- 🏆 Win and lose conditions

## 🛠️ Technologies Used

- Python 3
- Random module
- Time module

## ▶️ How to Run

Follow these steps to run the game on your computer:

### 1. Clone the repository

```bash
git clone https://github.com/rimi-sudo434/Mini_RPG_Game.git
```

### 2. Navigate to the project folder

```bash
cd Mini_RPG_Game
```

### 3. Run the RPG game

```bash
python rpg_game.py
```
## 🎯 How to Play

1. Enter your hero name.
2. Choose your action during battle:

   - `1` → Attack ⚔️
   - `2` → Heal 💚
   - `3` → Special Attack 🔥

3. Defeat the Dragon before your HP reaches zero.
4. Use strategy to survive and win the battle.

## 🎮 Example Gameplay

```
Enter your hero name: Alex

🔥 Welcome, Alex
⚔️ A wild Dragon appeared!

Your HP: 100
Dragon HP: 100

1. Attack
2. Heal
3. Special Attack

Choose your action: 1

You attacked the Dragon!
You dealt 20 damage.
```
